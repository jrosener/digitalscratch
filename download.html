Digital-Scratch v1.7.0 (November 2018)

Ubuntu GNU/Linux

Ubuntu packages for i386 and amd64. Please type the following commands in a terminal:

  • sudo add-apt-repository ppa:julien-rosener/digitalscratch
  • sudo apt-get update
  • sudo apt-get install digitalscratch

Check this page to run Digital-Scratch on GNU/Linux: how to

Debian Stable GNU/Linux

Debian Stable packages for i386 and amd64. Please type the following commands as root in a terminal:

  • apt-get install software-properties-common
  • wget -qO - http://www.digital-scratch.org/debian/julien.rosener@digital-scratch.org.gpg.key | apt-key add -
  • add-apt-repository "deb http://www.digital-scratch.org/debian/ stable main"
  • apt-get update
  • apt-get install digitalscratch
